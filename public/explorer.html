Tài khoản API Keys Đăng xuất
✅ Không cần auth

Chỉ cần account ID trong URL là gọi được. Chọn account bên dưới → mọi cURL tự fill ID.

Account ID là credential — đừng share ra ngoài.

👋 Hướng dẫn nhanh

  1. Tạo API key tại /api-keys.html → paste vào ô bên trái
  2. Thêm tài khoản Zalo tại / → chọn account ở sidebar
  3. Chọn method ở sidebar bên trái
  4. Đọc URL, Body, Fields, Response → copy cURL hoặc dùng client của bạn

Mọi response đều theo envelope { ok, data, meta }. Lỗi: { ok: false, error: { code, message } }.
