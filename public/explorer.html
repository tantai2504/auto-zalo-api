🔑 Cần API key
Mọi request cần header X-API-Key. Tạo key tại /api-keys.html.

👋 Bắt đầu

  1. Tạo API key tại /api-keys.html (lưu key — chỉ hiển thị 1 lần).
  2. Thêm tài khoản Zalo tại / rồi copy Account ID.
  3. Chọn nhóm tính năng bên trái → click method để xem cách gọi.
  4. Mọi response đều có dạng { ok, data, meta }.
