Chỉnh sửa

Dán token mới để cập nhật credentials cho tài khoản này. Server sẽ thử đăng nhập với token mới rồi thay thế credentials cũ.

Hoặc nhập trực tiếp z_uuid + zpw_sek

Listener

Khi bật, server lắng nghe event Zalo (message / reaction / undo / group_event ...) cho tài khoản này. Listener tự attach khi có WebSocket subscriber hoặc webhook URL.

Trạng thái listener:

Webhook

Khi listener bật, mọi event sẽ POST tới webhookUrl. Nếu có webhookSecret, body được ký HMAC-SHA256 — kiểm header X-Signature: sha256=<hex>.

Cookie session —

Đây là cookie session đã được giải mã. Dùng để debug hoặc xuất sang tool khác. Không chia sẻ ra ngoài.
