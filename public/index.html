Chỉnh sửa

Dán token mới để cập nhật credentials cho tài khoản này. Server sẽ thử đăng nhập với token mới rồi thay thế credentials cũ.

Hoặc nhập trực tiếp z_uuid + zpw_sek

Khi listener bật, mọi event Zalo (message / reaction / undo / group_event ...) sẽ POST tới webhookUrl. Nếu có webhookSecret, body được ký HMAC-SHA256 — kiểm header X-Signature: sha256=<hex>.
