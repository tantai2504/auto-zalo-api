Chỉnh sửa

Dán token mới để cập nhật credentials. Server sẽ thử đăng nhập bằng token mới rồi thay credentials cũ.

Hoặc nhập trực tiếp z_uuid + zpw_sek

Listener

Khi bật, server lắng nghe event Zalo (message / reaction / undo / group_event ...). Listener tự attach khi có WebSocket subscriber hoặc webhook URL.

Trạng thái:

Webhook

Khi listener bật, mọi event sẽ POST tới webhookUrl. Có webhookSecret → body được ký HMAC-SHA256 (header X-Signature).

Cookie session —

Cookie session đã giải mã. Dùng để debug. Không chia sẻ ra ngoài.
